# Global E-Commerce Sales Analysis (2025)

## Projenin Amacı
2025 yılına ait global e-ticaret satış verilerini analiz ederek pazar trendlerini, en kârlı ülkeleri ve ürün kategorilerini belirlemek.

## Kullanılan Teknolojiler
* **Veritabanı:** MySQL (Veri depolama ve karmaşık sorgular)
* **Programlama:** Python (Pandas, Matplotlib, Seaborn ile görselleştirme)
* **Editör:** VS Code & MySQL Workbench

## Temel Analizler ve Bulgular
1. **Pazar Liderleri:** USA ($206K+) ve UAE ($185K+) ciro bazında en güçlü pazarlardır.
2. **Zaman Trendi:** Satışlar Temmuz ayında zirve noktasına ulaşmıştır.
3. **Veri Kalitesi:** NULL değer, mükerrer kayıt ve matematiksel hata kontrolleri yapılarak verinin %100 doğruluğu teyit edilmiştir.

## Görselleştirme
Proje kapsamında oluşturulan grafikler, satışların coğrafi dağılımını ve zamansal değişimini net bir şekilde ortaya koymaktadır.
